# Meteorite System

The Meteorite System is the core of the Neon Survivor progression, allowing players to find, collect, and optimize tactical modules for their chassis.

## 1. Acquisition
Meteorites can be obtained by neutralizing hostiles during missions.

- **Drop Rate**: Basic hostile neutralization has a **1% base chance** to drop a meteorite.
- **No Grace Period**: Meteorites can drop starting from the first hostile neutralization.
- **Scaling**: Drop rates can be improved via Legendary Buffs and Matrix Overdrive enhancements.

## 2. Rarity & Progression
Meteorite rarity scales over time, with higher tiers becoming accessible as the mission duration increases.

| Minute | Rarity Pool Distribution (Anom / Rad / Abyss / Eter / Div / Sing) |
| :--- | :--- |
| 0-10 | 80% / 15% / 4.5% / 0.5% / 0% / 0% |
| 10-20 | 10% / 35% / 40% / 14.5% / 0.5% / 0% |
| 20-30 | 0% / 0% / 40% / 50% / 9.5% / 0.5% |
| 30-40 | 0% / 0% / 0% / 20% / 70% / 10% |
| 40+ | 0% / 0% / 0% / 0% / 60% / 40% |

## 3. Structural Integrity (Quality)
Each meteorite dropped has a static quality level that determines its base perk values and available ranges.

- **New (20%)**: Optimized integrity. Highest base perks.
- **Damaged (30%)**: Sub-optimal performance.
- **Broken (50%)**: Significant structural failure. Lowest base perks.

Players can use **Isotopes (Flux)** to repair meteorites (from Broken to Damaged, and Damaged to New) at the Recalibration Station.

## 4. Recycling & Salvage
Meteorites that are not installed in the Matrix can be recycled for **Recycling Dust**.

- **Salvage Value**: Higher rarities yield significantly more dust (increased by a 5x multiplier globally).
- **Quantum Scrapper**: This blueprint grants a 25% chance to double salvage yields.

## 5. Recalibration
At the Recalibration Station, players can spend Isotopes to:
- **Repair Quality**: Move a meteorite up a quality tier.
- **Reroll Types**: Randomize perk IDs for all unlocked slots.
- **Reroll Values**: Randomize the specific percentage values within the perk's range.

Costs for recalibration increase with the number of locked slots and the item's version.
