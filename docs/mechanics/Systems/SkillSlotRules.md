# Skill Slot Rules

This document describes the rules governing skill slots and the acquisition of legendary modules.

## 1. Maximum Skill Limit
The matrix module has a maximum capacity of **6 legendary skill slots**. 

## 2. Selection Restrictions
To ensure system stability and prevent placement conflicts, the following rules apply when selecting upgrades:

- **Empty Slots Available**: If the player has fewer than 6 legendary skills, the upgrade pool includes both new modules and level-ups for existing modules.
- **Maximum Slots Reached**: Once the player has filled all 6 slots, the upgrade pool is restricted. New legendary modules will no longer appear in the selection menu.
- **Level-Up Priority**: When the 6-slot limit is reached, only level-up options for currently equipped skills are presented to the player.
- **Max Level Reward**: If a player reaches the maximum 6-slot limit and all skills are at Level 4, boss kills will instead reward the player with **500 Dust**. This is indicated by a floating text popup over the player.

## 3. Level Progression
Legendary modules can be upgraded up to **Level 4**. After reaching Level 4, they are removed from the upgrade pool unless they can participate in a fusion.
