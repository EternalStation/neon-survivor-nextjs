# EPICENTER (DefEpi)

**Категория:** Defensive | **Арена:** 2 (пул: KineticBattery, DefPuddle, DefEpi, ChronoPlating)

## Перки по уровням

| Уровень | Перк |
|---------|------|
| 1 | Активный навык: канал шипов 10s. 70% замедление врагов, 25% MaxHP/sec. Кулдаун 30s |
| 2 | Во время канала: +50% снижение получаемого урона |
| 3 | На старте канала: 3s полная неуязвимость |
| 4 | Замедление до 80%, урон 35% MaxHP/sec. Кулдаун 30s |
| 5 | MAX LEVEL |

## Механика

**Активный навык:** при активации игрок иммобилизуется (`player.immobilized = true`) и создаётся область `epicenter`.

- Движение во время канала отменяет эффект (деактивирует шипы).
- Шипы наносят урон всем врагам в зоне: `enemy.maxHp × 0.25/sec` (L1–L3) или `0.35/sec` (L4).
- Враги в зоне замедляются на 70% (L1–L3) или 80% (L4).

**L2 — Снижение урона:**
```
reducedDmg *= 0.5   (50% снижение входящего урона во время канала)
```
Применяется в `handleEnemyContact`, проверяется: `if (player.immobilized && getHexLevel(state, 'DefEpi') >= 2)`.

**L3 — Временная неуязвимость:**
- `player.invincibleUntil = gameTime + 3.0` при активации.
- Весь входящий урон = 0 в течение 3 секунд.

**Кулдаун:** 30s base (масштабируется [Cooldown Reduction](../stats/cooldown-reduction.md)).

## Затронутые характеристики

DefEpi не изменяет PlayerStats напрямую. L2 применяет снижение урона inline при контактном уроне.
- [Снижение кулдауна](../stats/cooldown-reduction.md) — применяется к кулдауну Epicenter
