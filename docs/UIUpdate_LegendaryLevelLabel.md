# UI Update: Legendary Level Label

## Changes
- Updated the label for Level 4 Legendary modules in the Matrix module.
- Removed the "FUSABLE" tag and replaced it with "LVL 4" to keep the UI consistent with other levels.

## Context
Level 4 legendaries can be fused into Level 5 (MAX) versions, but the term "FUSABLE" was deemed redundant or inconsistent with the general level naming convention. 
Now all modules from level 1 to 4 show their level explicitly (`LVL 1`, `LVL 2`, `LVL 3`, `LVL 4`).
Level 5 modules continue to show `MAX`.
