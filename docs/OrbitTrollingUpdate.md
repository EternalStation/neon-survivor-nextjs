# Update: Orbit Assistant Trolling (Class Selection)

The assistant text for the class selection menu, triggered after 1 minute of inactivity, has been updated to be more "trolling" and sarcastic, as per user request.

## Changes:
- Modified all `reading` variant strings in `uiTranslations.ts` (English version).
- Texts are now more sarcastic, focusing on pilot procrastination and slow decision-making.
