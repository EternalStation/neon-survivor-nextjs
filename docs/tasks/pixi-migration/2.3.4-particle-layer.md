# Task 2.3.4 — Particle Layer

## Status: Done

## Dependencies: 2.1, 2.2

## Goal
Migrate particle rendering to PixiJS `ParticleContainer` for massive performance gain.

## Current Implementation
- Particles drawn as individual Canvas shapes (arc, rect)
- CPU particle cap: 700 void + 500 regular = 1,200 total

## PixiJS Implementation
- Use `ParticleContainer` — optimized for 10,000+ identical sprites
- Each particle type gets one texture (shard, spark, bubble, vapor, void)
- Remove CPU cap — GPU can handle 5,000-10,000+

```typescript
const particleContainer = new ParticleContainer(10000, {
  position: true,
  scale: true,
  tint: true,
  alpha: true,
  rotation: false,
})
```
