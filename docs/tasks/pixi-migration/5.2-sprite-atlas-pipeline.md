# Task 5.2 — Full Sprite Atlas Pipeline

## Status: Not Started

## Dependencies: Phase 2 (requires PixiJS), Task 2.2

## Goal
Create a complete sprite atlas pipeline for maximum draw call batching.

## Implementation

Use `@pixi/spritesheet` or texture packer:

1. At startup, render all enemy shapes via `Graphics` -> `RenderTexture`
2. Pack into atlas
3. All enemies become `Sprite` instances — one draw call for hundreds of enemies

```typescript
function generateEnemyAtlas(renderer: Renderer): Record<string, Texture> {
  const atlas: Record<string, Texture> = {}
  for (const shape of ALL_SHAPES) {
    for (const palette of ALL_PALETTES) {
      const g = new Graphics()
      drawEnemyShape(g, shape, palette)
      atlas[`${shape}_${palette.name}`] = renderer.generateTexture(g)
    }
  }
  return atlas
}
```

## Expected Result
- Draw calls reduced from 300-500+ to 10-20 per frame
- All entities of same type batched in single draw call

- [ ] All bullet types pre-rendered to atlas
- [ ] All particle types pre-rendered to atlas
- [ ] Player shapes pre-rendered to atlas
