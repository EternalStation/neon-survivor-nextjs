# Task 2.2 — Sprite Atlas Generation

## Status: Done

## Dependencies: 2.1

## Goal
Generate sprite atlases from procedural shapes for efficient GPU batching.

## Implementation

### Strategy: Start with Option B, upgrade to Option A

**Option B (quick start)**: Use PixiJS `Graphics` objects directly
- Replace `ctx.arc()` with `new Graphics().circle()`
- GPU-accelerated but not batched as efficiently as sprites

**Option A (target)**: Pre-rendered sprite atlas
- Render each enemy shape/palette/shell-stage to a texture atlas at startup
- Use `Graphics` -> `generateTexture()` to convert
- Batch all sprites in one draw call per layer

### Atlas Generation

```typescript
function generateEnemyAtlas(renderer: Renderer): Record<string, Texture> {
  const atlas: Record<string, Texture> = {}
  for (const shape of ALL_SHAPES) {
    for (const palette of ALL_PALETTES) {
      const g = new Graphics()
      drawEnemyShape(g, shape, palette)
      atlas[`${shape}_${palette.name}`] = renderer.generateTexture(g)
    }
  }
  return atlas
}
```

### Textures to generate
- Enemy shapes per palette (circle, triangle, square, diamond, pentagon + variants)
- Glow circles (8-10 sizes) for additive blending
- Bullet shapes per type
- Particle shapes (shard, spark, bubble, vapor)
