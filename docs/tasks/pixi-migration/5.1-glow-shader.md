# Task 5.1 — Custom Glow Shader

## Status: Not Started

## Dependencies: Phase 2

## Goal
Create a GPU-based glow filter to replace all `shadowBlur` usage with a single efficient shader.

## Implementation

Option A — Custom GLSL shader:
```glsl
// GlowFilter.frag
varying vec2 vTextureCoord;
uniform sampler2D uSampler;
uniform float strength;
uniform vec3 glowColor;

void main() {
  vec4 color = texture2D(uSampler, vTextureCoord);
  float glow = 0.0;
  for (int i = -4; i <= 4; i++) {
    for (int j = -4; j <= 4; j++) {
      vec2 offset = vec2(float(i), float(j)) * strength / 400.0;
      glow += texture2D(uSampler, vTextureCoord + offset).a;
    }
  }
  glow = glow / 81.0;
  gl_FragColor = color + vec4(glowColor * glow * (1.0 - color.a), glow * (1.0 - color.a));
}
```

Option B — Use `@pixi/filter-glow` package (pre-built solution).
