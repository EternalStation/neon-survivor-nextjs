# Task 1.3 — Bullet Interface Refactoring

## Status: Done

## Dependencies: None

## Goal
Break the monolithic `Bullet` interface (~40 fields) into composable sub-objects, preferably using discriminated unions.

## Target Structure

### Option A — Sub-interfaces

- **BulletCore** — id, ownerId, x, y, vx, vy, dmg, pierce, life, maxLife, isEnemy, size, color, hits
- **BulletVisual** — trails, isCrit, isVisualOnly
- **BulletSpecial** — isRing, isNanite, isMist, isShockwaveCircle, isSingularity, isTsunami, isHyperPulse
- **BulletStatusEffects** — slowPercent, freezeDuration, burnDamage
- **BulletTurret** — isTurretFire, turretLevel, turretVariant, isBomb, explodeRadius

### Option B — Discriminated Unions (preferred)

```typescript
type BulletType =
  | { kind: 'normal' }
  | { kind: 'ring'; radius: number }
  | { kind: 'nanite'; groupId: number }
  | { kind: 'shockwave'; level: number; maxSize: number }
  | { kind: 'singularity'; vortexState: number }
  | { kind: 'turret'; level: number; variant: string; isBomb: boolean }
```

This eliminates 15+ boolean flags and enables automatic type narrowing.

Same incremental approach as Tasks 1.1/1.2.
