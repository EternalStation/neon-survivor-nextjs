# Task 0.3 — Object Pool for Particles and Bullets

## Status: Not Started

## Dependencies: None

## Goal
Eliminate GC pressure from per-frame array allocation by introducing object pooling and swap-and-pop removal.

## Implementation

### 1. Create `packages/shared/logic/core/ObjectPool.ts`

```typescript
export class ObjectPool<T> {
  private pool: T[] = []
  constructor(private factory: () => T, private reset: (obj: T) => void) {}

  acquire(): T {
    return this.pool.length > 0 ? this.pool.pop()! : this.factory()
  }

  release(obj: T): void {
    this.reset(obj)
    this.pool.push(obj)
  }
}
```

### 2. Replace `filter()` array cleanup with swap-and-pop + pool release
