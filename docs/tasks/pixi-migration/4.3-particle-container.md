# Task 4.3 — ParticleContainer Integration

## Status: Not Started

## Dependencies: Phase 2 (requires PixiJS), Task 2.3.4

## Goal
Fully integrate PixiJS `ParticleContainer` with object pool for maximum particle performance.

## Implementation

```typescript
import { ParticleContainer, Sprite, Texture } from 'pixi.js'

const particleContainer = new ParticleContainer(10000, {
  position: true,
  scale: true,
  tint: true,
  alpha: true,
  rotation: false,
})

function spawnParticle(texture: Texture, x: number, y: number, color: number) {
  const sprite = particlePool.acquire()
  sprite.texture = texture
  sprite.position.set(x, y)
  sprite.tint = color
  particleContainer.addChild(sprite)
}
```

- Combined with object pool (Task 0.3) for zero GC pressure
