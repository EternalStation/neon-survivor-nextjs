# Task 5.3 — BitmapFont for Damage Numbers

## Status: Not Started

## Dependencies: Phase 2

## Goal
Replace `ctx.fillText()` with PixiJS `BitmapText` for floating damage/heal numbers.

## Current State
Floating numbers rendered with Canvas 2D `fillText()` — ~2ms/frame with 50+ numbers.

## Implementation

```typescript
import { BitmapFont, BitmapText } from 'pixi.js'

// At init: generate bitmap font
BitmapFont.from('DamageFont', {
  fontFamily: 'monospace',
  fontSize: 24,
  fill: '#ffffff',
  stroke: '#000000',
  strokeThickness: 2,
})

// Per floating number:
const text = new BitmapText({
  text: '-42',
  style: { fontFamily: 'DamageFont' }
})
```

## Expected Result
Text rendering: ~2ms/frame -> ~0.1ms/frame with 50+ floating numbers.

## Acceptance Criteria
- [ ] BitmapFont generated at startup
- [ ] All floating numbers use BitmapText
- [ ] Font supports digits, minus, plus, and common symbols
- [ ] Different colors for damage/heal/crit numbers
- [ ] Text rendering time <0.2ms with 50+ numbers
