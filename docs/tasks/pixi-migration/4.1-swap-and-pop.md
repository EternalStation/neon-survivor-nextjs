# Task 4.1 — Swap-and-Pop Array Removal

## Status: Not Started

## Dependencies: Phase 2 (benefits most with PixiJS)

## Goal
Replace `array.filter()` (allocates new array every frame) with in-place removal to eliminate GC pressure.

## Implementation

```typescript
function removeDeadInPlace<T>(
  arr: T[],
  isDead: (item: T) => boolean
): void {
  let write = 0
  for (let read = 0; read < arr.length; read++) {
    if (!isDead(arr[read])) {
      if (write !== read) arr[write] = arr[read]
      write++
    }
  }
  arr.length = write
}
```
