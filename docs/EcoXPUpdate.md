# EcoXP (Neural Harvest) Legendary Hex Balance Update

The per-kill bonuses for the EcoXP legendary hex (Neural Harvest) and its derived fusions (Xeno-Alchemist, Neural Singularity) have been adjusted to balance resource scaling.

## Changes:

| Tier | Pre-Update Value | Post-Update Value | Description |
| :--- | :--- | :--- | :--- |
| **LVL 1** | +0.1 XP | **+0.05 XP** | XP per kill bonus reduced |
| **LVL 2** | +0.05 Dust | **+0.02 Dust** | Dust per kill bonus reduced |
| **LVL 3** | +0.1 Flux | **+0.05 Flux** | Flux per kill bonus reduced |
| **LVL 4** | +0.1% XP | **+0.05% XP** | Percentage XP per kill bonus reduced |

## Files Modified:
- `src/logic/upgrades/LegendaryLogic.ts`: Updated bonus calculation logic.
- `src/lib/uiTranslations.ts`: Updated English perk descriptions.
