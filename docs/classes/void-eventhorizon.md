# Void `eventhorizon`

## Характеристики
| Параметр | Значение |
|----------|----------|
| Броня | +30% |
| Скорость передвижения | +10% |

---

## Способность: Void Singularity

**Триггер:** попадание снаряда + кулдаун завершён → сингулярность в точке попадания.

| Параметр | Значение |
|----------|----------|
| Кулдаун | 10 сек |
| Кулдаун с NEURAL_OVERCLOCK | 7 сек (×0.7) |
| Длительность | 3 сек |
| Радиус поля | 400px |
| Притяжение/кадр | `(0.66 + resonance × 0.85) × classCurseMult` |

### Урон в секунду по типу врага
| Тип | Условие | Урон |
|-----|---------|------|
| Обычный | >80px от центра | 50% maxHP/сек |
| Обычный | ≤80px (ядро) | мгновенное уничтожение |
| Сегмент червя | ≤80px (ядро) | мгновенное уничтожение |
| Элита / Редкий | — | 25% maxHP/сек |
| Босс | — | 10% maxHP/сек |

Покадровая формула: `dmg = percent × maxHp × step / 6`

Все враги в зоне получают флаг `voidAmplified = true`.
