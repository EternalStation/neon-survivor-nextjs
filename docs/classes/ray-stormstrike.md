# Ray `stormstrike`

## Характеристики
| Параметр | Значение |
|----------|----------|
| Урон | +40% |
| Скорость атаки | −20% |

---

## Способность: Orbital Strike

**Триггер:** попытка выстрела при готовом кулдауне — срабатывает параллельно с обычным выстрелом.

| Параметр | Значение |
|----------|----------|
| Кулдаун | 8 сек |
| Кулдаун с NEURAL_OVERCLOCK | 5.6 сек (×0.7) |
| Задержка удара | 0.3 сек (визуальный маркер + звук `lock-on`) |
| Урон | 150% от `player.dmg` |
| Радиус АОЕ | `100 × (1+resonance) × classCurseMult` px |
| Радиус поиска цели | 1000px |
| Выбор цели | случайный живой враг (`dead=false`, `isFriendly=false`, не `underground`, не в фазе смены) |
| Если врагов нет | удар в точку на 1000px по направлению прицела |

**АОЕ:** все враги в радиусе получают полный урон одновременно (нет деления).
