# Vortex `aigis`

## Характеристики
| Параметр | Значение |
|----------|----------|
| HP | +20% |
| Регенерация HP | +50% |

---

## Способность: Magnetic Vortex

**Триггер:** каждый выстрел игрока → проверка шанса для каждого уровня кольца независимо.

### Кольца
| Уровень | Радиус | Базовый шанс | Формула шанса |
|---------|--------|:------------:|---------------|
| Ring I  | 125px  | 100% | всегда |
| Ring II | 190px  | 15% | `0.15 × (1+resonance) × classCurseMult` |
| Ring III | 255px | 10% | `0.10 × (1+resonance) × classCurseMult` |
| Ring IV | 320px  | 5%  | `0.05 × (1+resonance) × classCurseMult` |

### Орбитальное движение снарядов (до слияния)
```
orbitAngle += 0.05 рад/кадр
x = player.x + cos(orbitAngle) × orbitDist
y = player.y + sin(orbitAngle) × orbitDist
```
Данные хранятся в `player.aigisRings[radius] = { count, totalDmg }`.

### Слияние / Распад
| Событие | Порог | Результат |
|---------|:-----:|-----------|
| Слияние | ≥200 снарядов | все снаряды → 1 объект `isRing: true` |
| Распад | <190 снарядов | 1 объект → 190 орбитальных снарядов |

### Урон кольца
- Зона поражения: аннулус ±10px от `ringRadius` (ширина 20px).
- Урон за попадание: `avgDmg = totalDmg / count`; при каждом попадании `count -= 1`, `totalDmg -= avgDmg`.
- Потеря при касании стены: **5 count/кадр**.
